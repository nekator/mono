## Vue

For general installation and configuration take a look at the [v-components](https://www.npmjs.com/package/@db-ui/v-components) package.

### Use component

```vue App.vue
<!-- App.vue -->
<script>
import { DBStack } from "@db-ui/v-components";
</script>

<template>
	<DBStack>
		<span>Test 1</span>
		<span>Test 2</span>
		<span>Test 3</span>
	</DBStack>
</template>
```
